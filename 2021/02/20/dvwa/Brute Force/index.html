3.2 注解

  • stripslashes(): 用于清理字符串中的反斜杠()
  • 延伸_addslashes()函数:
    php 会对所有的 GET，POST 和 COOKIE 数据自动运行 addslashes()!!
  • htmlspecialchars()函数:把预定义的字符转换为 HTML 实体。这些预定义的字符是&，”(双引号)，’（单引号），<(小于号), >(大于号)

3.3 漏洞分析

  • High 级别的代码加入了 Token，可以抵御 CSRF 攻击，同时也增加了爆破的难度，每次服务器返回的登陆页面中都会包含一个随机的 user_token 的值，用户每次登录时都要将 user_token 一起提交。服务器收到请求后，会优先做 token 的检查，再进行 sql 查询。

总结

  • Low:只是单纯的暴力破解，没有任何过滤
  • Mid:增加了过滤措施，防止了 sql 注入,而且加入了 sleep()函数，暴力破解时间增加
  • High:在 Low 的基础上，增加了 token 参数，如何获取 token 参数呢？可以使用 Python 脚本
