五章 给网页添加图像

  1. 浏览器是如何处理图像：

          从服务器获取'.html'文件-->浏览器显示html文件，发现需要图片-->继续从服务器获得。

  2. 区分图片的格式：

    • jpeg格式：照片和复杂图像
    • png和gif：单色图像、logo和几何图像

  3. 网页中的放置图片使用 元素，它是内联元素，所以添加图片后不会自动插入换行， 元素还有有’width’和’height’属性来设置图片的大小
