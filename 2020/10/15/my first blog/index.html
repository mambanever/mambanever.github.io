Mac下基于GitHub的Hexo博客搭建及更换主题。（命令基于root权限下）

一、node.js安装（用于生成静态页面）

  1. 直接通过官网下载https://link.zhihu.com/?target=https%3A//nodejs.org/en/，并且安装。

  2. 通过node -v来检查是否安装成功。

二、Git安装及配置（将本地内容提交到GitHub（已注册）上）

  1. 在终端输入brew install git来下载，git -v检查安装成功（Mac下brew安装详见）

  2. ssh配置关联GitHub仓库详见https://blog.csdn.net/linhai1028/article/details/81436825?biz_id=102&utm_term=Mac%E4%B8%8Bgit&utm_medium=distribute.pc_search_result.none-task-blog-2~all~sobaiduweb~default-3-81436825&spm=1018.2118.3001.4187

三、hexo安装（通过npm）

  1. 更新安装源：npm config set regitory https://regitory.npm.taobao.org（如不更换，安装会卡住）
  2. 验证是否更新成功：npm config get regitory
  3. 安装cnpm：npm install -g cnpm --registry=https://registry.npm.taobao.org
  4. 安装hexo：npm install -g hexo-cli

四、hexo初始化

  1. mkdir blog
  2. cd blog/
  3. hexo init//hexo初始化
  4. hexo g生成静态页面,hexo s之后即可访问http://localhost:4000（不要停止终端）

五、hexo部署到GitHub

  1. 进入blog文件夹，可以看到_ config.yml文件(blog配置文件)。

六、完成
