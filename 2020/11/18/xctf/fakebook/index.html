|阅读量:

知识点：

  • 1.sql 注入空格绕过，union select 会被 WAF 检测到，所以使用 /**/ 来绕过，基本注入
  • 2.反序列漏洞的基本代码审计
  • 3.注重页面中的各种提示性语句。

两个解法大佬就是大佬，思路清晰
user(),load_file()巧妙使用。。。

正确的思路流程应当为，得到备份文件，明确 getBlogContents 函数通过一个 URL 获取内容，并且 URL 由类实例化对象的 blog 参数提供。然后在构造 userinfo 对象序列化发现页面可以正常返回之后，就尝试修改序列化中的 blog 参数，发现页面中 iframe 的 src 暴露信息，进而想到可以使用伪协议读取 flag，并让 src 显示。
